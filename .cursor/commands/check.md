# 检查项目质量

## 描述
全面检查 Godot 项目的代码质量、语法错误和最佳实践。

## 使用方式
在 AI 对话中输入：`/check`

## 功能
- 运行 Godot 语法检查
- 分析代码质量和结构
- 检查最佳实践遵循情况
- 提供改进建议

## AI 交互流程
1. 运行 Godot 语法检查
2. 分析项目结构和代码
3. 检查类型注解使用
4. 检查命名规范
5. 检查错误处理
6. 提供详细的改进建议
7. 生成质量报告

## 关键命令

### 语法检查
```bash
# 运行 Godot 语法检查（最简单有效的方法）
godot --headless --quit
```

## 预期结果
- 语法错误列表
- 代码质量评估
- 改进建议
- 最佳实践检查结果
